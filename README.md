
# ASC Atmiya Student Community

An android App. Where student's of Atmiya connect together at same platform
They can do lot's of things like 
- [ ] Question Answers.
- [ ] Create Groups
- [ ] Private Chat
- [ ] Forum
- [ ] Study document
- [ ] Projects & assignments
- [ ] Announce new events or many more
